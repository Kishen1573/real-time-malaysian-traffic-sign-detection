# 🚀 Real-Time Malaysian Traffic Sign Detection System

## 📌 Overview
The **Real-Time Malaysian Traffic Sign Detection System** is an intelligent computer vision project designed to enhance road safety and support smart transportation systems.

It uses **YOLOv8 object detection** to identify Malaysian traffic signs in real time and a **CNN-based classifier** for speed limit recognition. The system processes video input, generates live detections, and provides instant alerts to drivers.

This project demonstrates practical applications of **AI in autonomous driving assistance and traffic monitoring systems**.

---

## 🎯 Key Features
- 🔍 Real-time traffic sign detection using YOLOv8
- 🚗 Speed limit classification using CNN model
- ⚡ Live video processing with frame-by-frame analysis
- 🚨 Instant alert system for detected traffic signs
- 📊 Detection logging for analysis and tracking
- 🧠 Optimized for Malaysian traffic sign dataset
- 🎥 Supports video input for testing and simulation

---

## 🧠 Tech Stack
- Python
- YOLOv8 (Ultralytics)
- Convolutional Neural Networks (CNN)
- OpenCV
- NumPy
- PyTorch

---

## 🏗️ System Workflow
1. Input video is processed frame by frame  
2. YOLOv8 detects traffic signs in real time  
3. Detected signs are cropped and analyzed  
4. CNN classifies speed-related signs  
5. System triggers alert based on detection  
6. Results are logged for review and evaluation  

---

## 📊 Model Highlights
- Trained on Malaysian traffic sign dataset  
- Supports multiple sign categories (speed limit, warning, etc.)  
- Optimized detection for real-world road scenarios  
- Combines detection + classification pipeline for higher accuracy
